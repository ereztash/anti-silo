ANTI-SILO FOR WINDOWS

Preferred option: download Anti-Silo-Setup.exe, open it, and follow the installer.
It creates an Anti-Silo shortcut for you.

Portable option:
1. Extract this folder anywhere on your computer.
2. Double-click Anti-Silo.exe.
3. Choose "בדוק את הקבצים שלי".

You do not need Python, GitHub Desktop, a terminal, or an account.

Anti-Silo works locally on your computer. It does not upload your files.

If Windows asks whether to run this new unsigned app, verify that you downloaded
it from https://github.com/ereztash/anti-silo/releases before selecting Run anyway.
